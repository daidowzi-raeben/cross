1) 코드

- 마크업

  • 마크업은 bem규칙을 원칙으로 한다
  • 개발에 필요한 부분은 개발자용 [D] 주석을 추가할 것
    • /* [D] 메시지 */
// 마크업 <div class="test"> /* [D] 활성화시 test-a에 is_active클래스 추가 */ <div class="test-a is_active"></div> <div class="test-b"></div> </div>
// scss .test{ &-a{} &-b{} }

- 클래스

  • 클래스명 사용 시에는 하이픈(-)을 사용할 것 예외: 활성화 시에는 is_active 로 통일
  • 공통클래스 참고할 것 (파일명: common.scss)
